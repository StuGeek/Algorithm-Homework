# 移动物品

小明将n个物品放置在X个箱子中，给出每个物品存在第几个箱子。
我们可以进行两种操作：
* 1.将第 i 个物品向左或者右移动 2 个箱子，cost为 0。
* 2.将第 i 个物品向左或者右移动 1 个箱子，cost为 1。

返回将所有物品放在同一箱子所需要的最小cost.<br/>
输入T为测试次数
1000>n>1,1<X<10<br/>
input:<br/>
T   //输入的测试样例数量<br/>
N   //物品的数量<br/>
n_1,n_2... //每个物品在第几个箱子<br/>
实例：<br/>
input:<br/>
1<br/>
2<br/>
2 2<br/>
output:<br/>
0<br/>

input:<br/>
2<br/>
2<br/>
3 3<br/>
3<br/>
2 3 3<br/>
output:<br/>
0<br/>
1<br/>
