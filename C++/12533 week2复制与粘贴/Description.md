# week2复制与粘贴

一开始在一个记事本上只有一个字符 'A'。你每次可以对这个记事本进行Ctrl+C或者Ctrl+V：</br>

Ctrl+C(复制全部) : 你可以复制这个记事本中的所有字符(部分的复制是不允许的)。</br>
Ctrl+V (粘贴) : 你可以粘贴你上一次复制的字符。</br>
给定一个数字 n 。你需要使用最少的操作次数，在记事本中打印出恰好 n 个 'A'。输出能够打印出 n 个 'A' 的最少操作次数。</br>

输入：</br>
T   //测试样例总数</br>
n ....   //T 个n </br>
示例：</br>
input:</br>
1</br>
3</br>
output:</br>
3</br>
